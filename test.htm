Fairness in a real social network.

Καλαμπόκης Γεώργιος-Θεόδωρος, Α.Μ : 423 Σέχαϊ Φατιόν, Α.Μ : 428

December 18, 2019

Τα κοινωνικά δίκτυα είναι ένα σύνολο αλληλεπιδράσεων και διαπροσωπικών σχέσεων. Ο όρος σήμερα χρησιμοποιείται επίσης για να περιγράψει ιστοσελίδες οι οποίες επιτρέπουν την διεπαφή ανάμεσα στους χρήστες, π.χ. με σχόλια, φωτογραφίες[ορολογία από wikipedia]. Τα online κοινωνικά δίκτυα έχουν επηρεάσει πάρα πολύ τις ζωές των ανθρώπων. Στα κοινωνικά δίκτυα συνήθως δεν εμφανίζεται δικαιοσύνη (fairness). Ένα παράδειγμα του fairness απο την καθημερινή μας ζωή, προκύπτει απο τα αποτελέσματα ενός ερευνητικού πειράματος του "AdFisher" , στο οποίο κατέληξαν στο συμπέρασμα ότι στις διαφημίσεις με θέμα τις πιο καλά αμειβόμενες θέσεις εργασίες τείνουν να συμμετέχουν κυρίως άντρες. Το Fairness είναι μια μετρική που χρησιμοποιείται και στα κοινωνικά δίκτυα με πολλούς τρόπους. Για παράδειγμα, στη περίπτωση μας, θα χρησιμοποιήσουμε το Github για να μετρήσουμε την δικαιοσύνη. Πιο συγκεκριμένα θα εξετάσουμε το φύλο από πολλούς χρήστες οι οποίοι συνδέονται μεταξύ τους στο Github.

Τα βήματα που θα ακολουθήσουμε περιγράφονται παρακάτω: Αρχικά, θα συλλέξουμε δεδομένα σχετικά με συνδέσεις χρηστών στο GitHub, χρησιμοποιώντας τις κλήσεις συστήματος (API) που προσφέρει για την συλλογή δεδομένων. Για να το επιτύχουμε αυτό θα εκτελέσουμε την συνάρτηση json.dump(“https://api.github.com/users/octocat”,outfile) από την οποία θα κρατήσουμε τις παρακάτω παραμέτρους για κάθε χρήστη("login", "id", "followers_url", "following_url", "name"). Στην συνέχεια θα αναλύσουμε τις συνδέσεις μεταξύ φίλων σε αυτό και θα βρούμε το φύλο με κατάλληλο λογισμικό. Θα εξάγουμε στατιστικά συμπεράσματα σχετικά με το φύλο τους. ************************************TOP! ***************************************

Το γράφημα μας θα είναι κατευθυνόμενο καθώς θα υπάρχουν following/followers συνδέσεις για κάθε χρήστη. Για τη δημιουργία του γραφήματος θα θεωρήσουμε τυχαία ως πρώτο κόμβο έναν χρήστη ο οποίος έχει σχετικά μεγάλο degree. Ο λόγος που ξεκινάμε από έναν κόμβο με σχετικά μεγάλο degree είναι για να μπορέσουμε να επεκτείνουμε το γράφημα μέσω των ακμών του. Έπειτα, λαμβάνοντας υπόψην τα άτομα τα οποία ακολουθεί και τον ακολουθούν, θα εισάγουμε και αυτά τα άτομα σαν κόμβους (με σειρά FIFO, BFS) και τις αντίστοιχες ακμές (συνδέσεις: following-followers ). Θα σταματήσουμε αφού πάρουμε ικανοποιητικό αριθμό από κόμβους.

***************************************ΔΙΑΧΥΣΗ!!***********************************************

Κατά την υλοποίηση του project εικάζουμε ότι θα εμφανιστούν κάποιες προκλήσεις-δυσκολίες. Παραθέτουμε παρακάτω μερικές απο αυτές: Η εύρεση των κατάλληλων για την περίπτωσή μας δεδομένων Το καθάρισμα των δεδομένων. Η είσοδος που λαμβάνει ο αλγόριθμος για την εξαγωγή του φύλου, υπάρχει πιθανότητα να μην δίνει τόσο καλά αποτελέσματα (λόγω ονόματος).

Τέλος, ως αποτέλεσμα του project αναμένουμε, να μην υπάρχει fairness στο δίκτυο, δηλαδή οι άνδρες να είναι περισσότεροι απο τις γυναίκες.
