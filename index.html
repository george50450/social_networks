Astract

Social networks are a set of interactions and interpersonal relationships. The term is also now used to describe websites that allow the interface between users, e.g. with comments, photos etc. The main goal of this paper is to measure fairness in a real social network, in particular Github, and how it affects the way information is being diffused between the users. Fairness is a metric used in social media in many ways. As fairness in our approach is defined the percentage of users gender. Firstly, we simulate the social network as a graph. The graph contains users as nodes and relationships (following - followers) as edges, which means that our graph is directed. In order to create our graph, we collected the aforementioned data using the API of GitHub. Online social networks influence people’s lives in many ways, so the existence of fairness is important. For this reason, we decided to measure it in male-dominated network. Comparing the percentages of gender from the world and the Github, we realized that there was a huge difference between them. During the diffusion, as we tested the nodes affected by information, we found out that the percentages of nodes that were affected, were corresponding to the original percentages of the data.

Fairness

Πρόκειται για μια μετρική η οποία χρησιμοποιείται στα κοινωνικά δίκτυα με ποικίλους τρόπους. Στη περίπτωση μας, θα χρησιμοποιήσουμε ως κοινωνικό δίκτυο το Github για να μετρήσουμε την δικαιοσύνη.

Our approach

How to measure Parity Fairness; Θα εξετάσουμε το φύλο από πολλούς χρήστες οι οποίοι συνδέονται μεταξύ τους στο Github. Αρχικά θα συλλέξουμε δεδομένα σχετικά με συνδέσεις χρηστών στο GitHub, χρησιμοποιώντας τις κλήσεις συστήματος (API) που προσφέρει για την συλλογή δεδομένων. Στην συνέχεια θα βρούμε το φύλο των κόμβων με κατάλληλο λογισμικό. Απο τους χρήστες-κόμβους που θα έχουμε στο γράφημα βρίσκουμε τους top με βάση τα stars που έχουν στα projects τους ή με βάση τα fork που έχουν συνολικά τα repositories του κάθε χρήστη. Με αυτό τον τρόπο θα τους κατατάξουμε. Έπειτα, θα εξάγουμε στατιστικά σχετικά με το φύλο και την κατάταξη. Θα ξεκινήσουμε από έναν κόμβο ο οποίος έχει σχετικά μεγάλο degree. Έπειτα, λαμβάνοντας υπόψην τα άτομα τα οποία ακολουθεί και τον ακολουθούν, θα εισάγουμε και αυτούς (FIFO, BFS) και τις αντίστοιχες ακμές μεταξύ τους (following-followers). Θα σταματήσουμε αφού πάρουμε ικανοποιητικό αριθμό από κόμβους. Θα μεταδώσουμε μια πληροφορία από έναν μικρό αριθμό κόμβων, (π.χ. κάποιους με μεγάλο degree). Σκόπος είναι αυτή η πληροφορία να μεταδοθεί σε όσο το δυνατόν περισσότερες κατηγορίες ομάδων έχουμε. Δε μας ενδιαφέρει ποια θα είναι η πληροφορία, αλλά τους κόμβους που θα “επηρεάσει/προσεγγίσει”.
